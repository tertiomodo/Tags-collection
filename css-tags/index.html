    21. Свойства overflow: visible/hidden/scroll/auto; - Свойства отвечающие за поведение такста в блоке (Подробная информация в CSS).

    overflow: visible; - свойство по умолчанию, когда текст игнорирует высоту блока.
    overflow: hidden; - свойство котрое обрезает все что выходит за пределы границ блока.
    overflow: scroll; - свойство добавляет горизонтальный и вертикальный скроллы в контейнер, при этом они будут показываться даже если текста нет.
    overflow: auto; - свойство которое вы видите, это самый оптимальный вариант для добавления скролла в блок, тк свойство добавляет скролл только при необходимости.
    overflow-x: scroll; - свойство позволяет управлять скроллом по оси X, те горизонтальной оси.
    overflow-y: scroll; - свойство позволяет управлять скроллом по оси X, те вертикальной оси.

    22. Свойство z-index:
    - Позволяет управлять плановостью элементов, например, в коде красный квадрат расположен первее в потоке, за ним синий, соответственно браузер отрисует красный а потом сверху синий, но свойство z-index: 1 для красного позволяет "вытащить" его на передний план.
    - Принимает отрицательные значения, допустим, у нас нет возможности сдвинуть синий квадрат выше, который находится на втором плане с z-index: 0, но можно использовать z-index: -1 для красного, что соответственно сделает синий квадрат выше. (Не рекомендуется к использованию)
    - Работает для любого свойства position.

    42. Базовые правила типографики.

    Используйте стандартные шрифты

    Лучший выбор при использовании шрифтов - стандартные шрифты. Например Arial, Verdana, Tahoma, Times New Roman и так далее. Большинство из этих шрифтов уже есть в компьютере пользователей, они привычны и легко читаются.

    Используйте не больше двух шрифтов на странице

    Это поможет пользователю быстрее ориентироваться на странице. Стандартным случаем является использование одного шрифта для заголовков и другого для текста.

    Используйте достаточный размер шрифта

    Это понятие двойственное, но не следует использовать основной текст со шрифтом меньше 14 пикселей. Если размер шрифта будет меньше, то чтение с мобильных устройств будет пыткой.

    Используйте достаточный межстрочный интервал

    Самым распространённым межстрочным интервалом является значение в 150% от размера шрифта. Если текст размером 14 пикселей, то межстрочный интервал желательно установить не менее 21 пикселя.

    Выравнивайте текст по левому краю

    Не стоит писать весь текст с выравниванием по центру или правому краю. При чтении таких текстов глаза устают намного быстрее, так как глаз теряет точку опоры, откуда начинается текст. Используйте нестандартное выравнивание только для небольших участков текста (например, заголовки или цитаты).

    45. Переменные - используются если мы заранее знаем стили которые будут использоваться в большом количестве блоков на нашем сайте

    :root {
      --main-color: #000000;
    }

    .new_block {
      background-color: var(--main-color);
    }

    .left_sidebar {
      background-color: var(--main-color);

    48. Градиенты. Ссылка с подробным разбором данного свойства

    В CSS существуют два вида граниентов, линейный (linear-gradient) и радиальный (radial-gradient)

    Линейные градиенты linear-gradient

    Линейный градиент создается с помощью двух и более цветов, для которых задано направление, или линия градиента.

    Синтаксис

    background: linear-gradient(угол / сторона или угол наклона с помощью ключевого слова (пары ключевых слов), первый цвет, второй цвет и т.д.); 
    
        .gradient {
          width: 200px;
          height: 200px;
          background-image: linear-gradient(180deg, wheat, crimson);
        }

    Для неравномерного распределения цветов указывается начальная позиция каждого цвета через точки остановки градиента (color stops). Точки остановки задаются в %, где 0% — начальная точка, 100% — конечная точка, например:

    background: linear-gradient(180deg, crimson 20%, black 50%, tomato 80%);

    Для чёткого распределения цветных полос каждый последующий цвет нужно начинать с точки остановки предыдущего цвета:

    background-image: linear-gradient(wheat 25%, crimson 25% 70%, navy 25%);

    Направление градиента - задается при помощи указания угла наклона линии градиента и измеряется в градусах или при помощи ключевых слов to top, to right, to bottom, to left, удобнее и приоритетнее использовать градусы.

    background-image: linear-gradient(45deg, wheat, crimson);

    Пример линейного градиента в оформлении рамки для текста:

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Iure, fugiat molestias inventore reiciendis corrupti sed, quos nobis ab ea ut omnis, repudiandae sunt! Sequi ab cupiditate consectetur iusto aperiam similique.

    Радиальные градиенты radial-gradient

    Радиальный градиент отличается от линейного тем, что цвета выходят из одной точки (центра градиента) и равномерно распределяются наружу, рисуя форму круга или эллипса.

    Синтаксис:

    background: radial-gradient(форма градиента / размер / позиция центра, первый цвет, второй цвет и т.д.);

    Форма градиента определяется ключевыми словами circle или ellipse. Если форма не задана, по умолчанию радиальный градиент принимает форму эллипса. Позиция центра задаётся с помощью ключевых слов at top, at right, at bottom, at left, at center или at Npx/N%/Nem. Если позиция центра не задана, используется значение по умолчанию at center.

    background-image: radial-gradient(circle at top, wheat, black);

    Размером радиального градиента можно управлять используя %, em или px. Первое значение задает ширину эллипса, второе — высоту.

    background-image: radial-gradient(50% 20%, wheat, black);

    55. Ключевые значения в CSS свойство: initial, inherit, unset, revert.

    У каждого стиля есть начальные свойства которые никак не связаны с браузером. Ключевые значения позволяют либо сбросить все до начальных значений, либо наследовать стили от родителя.

    CSS правило состоит из: 

      
  .селектор {
    свойство: значение;
  }
    • initial - Возвращает стили к своему самому начальному значению.
    • inherit - Дочерний элемент наследует стили от родительского элемента.
    • unset - Работает в зависимости от типа свойства к которому применяется. В CSS есть два типа свойства, наследуемые (например color) и не наследуемые (например display), если свойство наследуемое, то unset сработает как inherit, а если свойство не наследуется, то unset превращается в initial.

      Возникает вопрос, а зачем нам свойство которое переопределяется либо в одно, либо в другое? Представим ситуацию что у элемента нужно сбросить все стили до начального значения, для этого мы можем каждому свойству задавать либо inherit либо initial, а можем применить свойство all: unset; которое сбросит все стили присутствующие у элемента до начальных значений.

    • revert - Аналог свойства unset, только оно сбрасывает стили не к начальным значениям, а к браузерным по умолчанию. (Является рекомендуемым, тк сбрасывает стили к привычным, браузерным).

    58. Свойство transition - Отвечает за плавность анимации. Ссылка с подробным разбором данного свойства

    • transition-property: color; - В значение вписывается то свойство, которое хотим анимировать. Данное свойство имеет следующие значения:
      • property name - Выборочно указываем какое свойство мы хотим анимировать
      • all - Для анимации выбираются все свойства объекта, что может затормаживать сайт
      • none - Сброс transition
    • transition-duration: 0.5s; - В данном свойстве задается длительность анимации в секундах или миллисекундах, например, 2s или 5ms
    • transition-timing-function: ease-in-out; - Является функцией которая управляет скоростью самой анимации. Для того чтобы создать тайминг используются ключевые слова которые подвязаны к кривой Безье, которая и отвечает за темп анимации, этих слов много, подробнее можно прочитать тут, а мы же воспользуемся одним из них
    • transition-delay: 0.2s; - Задержка перед началом анимации, выставляется так же в секундах и миллисекундах
    • transition: color 0.5s ease-in-out 0.2s; - Все свойства выше в одну строку! Рекомендация к использованию

    Указывая transition начальному элементу, к примеру у нас есть классы link, link:focus, link:hover и link:active, и если мы пропишем transition элементу link, без псевдоклассов, то анимация будет работать в обе стороны (когда наводим курсор и когда убираем).

    Действия которые мы совершаем для того, чтобы по наведению на текст, цвет плавно превращался в красный: 

    
      .text {
        font-size: 25px;
      }
      
      .text:hover { (1)
        color: tomato; 
        transition-property: color; (2)
        transition-duration: 0.5s; (3)
        transition-timing-function: ease-in-out; (4)
        transition-delay: 0.2s; (5)
        --------------------------------------------------
        transition: color 0.5s ease-in-out 0.2s; (6)
      }
    1. К селектору .text применяем псевдокласс :hover для того чтобы цвет менялся при наведении
    2. Указываем какое свойство мы хотим анимировать (в нашем случае мы анимируем цвет при наведении), используем для этого свойство transition-property: color; мы сообщаем браузеру "я указал цвет по наведению, теперь примени мне анимацию только к указанному цвету".
    3. Указываем время длительности анимации. Указывать можно в секундах и миллисекундах
    4. При помощи ключевого слова выбираем скорость анимации, все ключевые слова подвязаны к кривой Безье, которая представляет из себя график функции по которму движется наша анимация
    5. Выставляем задержку перед началом анимации. Указывать можно в секундах и миллисекундах
    6. Или можем указать все что мы писали выше, но в одну строку!
