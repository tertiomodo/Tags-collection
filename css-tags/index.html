    21. Свойства overflow: visible/hidden/scroll/auto; - Свойства отвечающие за поведение такста в блоке (Подробная информация в CSS).

    overflow: visible; - свойство по умолчанию, когда текст игнорирует высоту блока.
    overflow: hidden; - свойство котрое обрезает все что выходит за пределы границ блока.
    overflow: scroll; - свойство добавляет горизонтальный и вертикальный скроллы в контейнер, при этом они будут показываться даже если текста нет.
    overflow: auto; - свойство которое вы видите, это самый оптимальный вариант для добавления скролла в блок, тк свойство добавляет скролл только при необходимости.
    overflow-x: scroll; - свойство позволяет управлять скроллом по оси X, те горизонтальной оси.
    overflow-y: scroll; - свойство позволяет управлять скроллом по оси X, те вертикальной оси.

    22. Свойство z-index:
    - Позволяет управлять плановостью элементов, например, в коде красный квадрат расположен первее в потоке, за ним синий, соответственно браузер отрисует красный а потом сверху синий, но свойство z-index: 1 для красного позволяет "вытащить" его на передний план.
    - Принимает отрицательные значения, допустим, у нас нет возможности сдвинуть синий квадрат выше, который находится на втором плане с z-index: 0, но можно использовать z-index: -1 для красного, что соответственно сделает синий квадрат выше. (Не рекомендуется к использованию)
    - Работает для любого свойства position.

    42. Базовые правила типографики.

    Используйте стандартные шрифты

    Лучший выбор при использовании шрифтов - стандартные шрифты. Например Arial, Verdana, Tahoma, Times New Roman и так далее. Большинство из этих шрифтов уже есть в компьютере пользователей, они привычны и легко читаются.

    Используйте не больше двух шрифтов на странице

    Это поможет пользователю быстрее ориентироваться на странице. Стандартным случаем является использование одного шрифта для заголовков и другого для текста.

    Используйте достаточный размер шрифта

    Это понятие двойственное, но не следует использовать основной текст со шрифтом меньше 14 пикселей. Если размер шрифта будет меньше, то чтение с мобильных устройств будет пыткой.

    Используйте достаточный межстрочный интервал

    Самым распространённым межстрочным интервалом является значение в 150% от размера шрифта. Если текст размером 14 пикселей, то межстрочный интервал желательно установить не менее 21 пикселя.

    Выравнивайте текст по левому краю

    Не стоит писать весь текст с выравниванием по центру или правому краю. При чтении таких текстов глаза устают намного быстрее, так как глаз теряет точку опоры, откуда начинается текст. Используйте нестандартное выравнивание только для небольших участков текста (например, заголовки или цитаты).

    43. Медиазапросы, правило @media.

    Какие правила указываются внутри конструкции @media? Абсолютно любые! Думайте об этом, как о новом CSS файле, со своими селекторами, свойствами. Например, для определённого условия нужно изменить цвет текста элементов с классом .text на чёрный.

    Представим, что нужно изменить цвет текста и фона блока ниже, если ширина viewport меньше или равно 700px. Тогда указывается правило max-width: 700px (Подробная информация в CSS). Cвойство сработает, если ширина viewport 700 пикселей или меньше:

    Text

    В медиа запросах есть специальное условие orientation, которые необходимо для того чтобы определять как именно пользователь держит телефон горизонтально или вертикально, свойство может иметь одно из двух значений:

    • portrait - портретная ориентация устройства. Высота экрана больше, чем его ширина
    • landscape - альбомная ориентация устройства. Высота экрана меньше, чем его ширина

    Например, пусть для портрентной ориентации текст будет белым, а для альбомной чёрным: 

    
      @media (orientation: portrait) {
        .text {
          color: #ffffff;
        }
      }
      
      @media (orientation: landscape) {
        .text {
          color: #000000;
        }
      }

    Для того, чтобы задать анимацию, используется правило @keyframes и название анимации. Далее, внутри @keyframes необходимо указать различные временные промежутки, в которых будут происходить изменения.     После того как мы задали инструкцию для анимации, необходимо прмиенить созданную нами анимацию к тексту, блоку или какому-либо элементу. Для этого используется правило animation (Подробная информация в CSS) с тремя основными значениями:

    • Название анимации
    • Длительность анимации. Указывается в секундах (1s, 2s, 3s и так далее)
    • Бесконечная анимация или нет (если бесконечная, то указывается параметр infinite)

    45. Переменные - используются если мы заранее знаем стили которые будут использоваться в большом количестве блоков на нашем сайте

    Переменная создаётся с помощью конструкции --имя-переменной. Имя переменной вы можете выбрать сами. Для примера создадим переменную --main-color, которая будет содержать базовый цвет страниц. Пускай он будет чёрным:

    --main-color: #000000;

    Для создания глобальной переменной её нужно указать в специальной конструкции :root. Обычно это делается в самом начале CSS файла: 

    
    :root {
      --main-color: #000000;
    }

    Теперь можно использовать переменную в любой части нашего CSS-кода. Это делается с помощью специальной конструкции var(--имя-переменной). 

    
    :root {
      --main-color: #000000;
    }

    .new_block {
      background-color: var(--main-color);
    }

    .left_sidebar {
      background-color: var(--main-color);
    }

    Если мы хотим установить другой оттенок чёрного цвета, достаточно изменить только значение переменной --main-color, и все изменения автоматически применятся к блокам с классами .news_block и .left_sidebar.
